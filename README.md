# Archives-Projects

This repository contains archived projects that were created for various purposes over time. Below is the directory structure and a brief description of the contents.

## Repository Structure

- **/Academic Projects**: Projects and assignments completed for academic purposes.
- **/General Block Code Projects**: Various code snippets, block code, and smaller experimental projects.
- **/Partially Completed or Failed Projects**: Ideas and projects that were started but not fully completed or abandoned.
- **/Real Completed Projects**: Fully functional and completed projects.

---

## Real Completed Projects

This directory contains larger, completed projects:

### 1. Browser Search Automation Script
A set of multithreaded Python scripts designed to automate web searches (e.g., using a list of general search queries to automate browser activity). Includes configurable time durations and queries.

### 2. VoxSys (Upgraded to Neuro Intelligence)
The initial version of a voice/system intelligence project, which was later upgraded and refactored into Neuro-Intelligence.

### 3. Neuro-Intelligence Backup
The continuation and upgraded version of the VoxSys project. This project includes core functionalities, model creation scripts, and a prompt-based working model (e.g., FLAN T5).
